# Premium Calculator - 美しくスマートな簡易電卓

グラスモフィズム（ガラス質感）デザインを採用した、モダンでプレミアムなWeb電卓アプリケーションです。  
HTML, CSS, JavaScript のみで構成されており、GitHub Pages での公開に完全対応しています。

---

## ✨ 主な機能

- **美しくダイナミックなデザイン**: グラスモフィズム質感のカード、背後の流動的なグラデーションオーブ、滑らかなホバー＆クリックアニメーション。
- **高精度な計算ロジック**: JavaScriptの浮動小数点演算特有の丸め誤差（例: `0.1 + 0.2`）を適切にハンドリング。
- **計算履歴機能**:
  - 過去の計算履歴（最大20件）を自動保存（ブラウザの `localStorage` を使用するためリロード後も維持）。
  - サイドパネルで履歴を表示し、クリックすることで過去の計算結果を現在の入力欄に呼び出せます。
- **キーボード操作対応**: 数字入力、四則演算、Enter（等号）、Backspace（一文字消去）、Escape（クリア）などのキーボードショートカットに対応。
- **レスポンシブデザイン**: スマートフォン、タブレット、PCなど、あらゆるデバイスで最適に表示されます。
- **アクセシビリティ**: 適切なARIA属性を付与し、フォーカスリングや操作説明も完備。

---

## ⌨️ キーボードショートカット一覧

| キー | 動作 |
| --- | --- |
| `0` - `9` | 数字の入力 |
| `.` | 小数点の入力 |
| `+`, `-`, `*`, `/` | 四則演算（`+`, `−`, `×`, `÷`）の選択 |
| `Enter` または `=` | 計算の実行（`=`） |
| `Backspace` | 1文字消去（`DEL`） |
| `Escape` または `c` / `C` | オールクリア（`AC`） |
| `%` | パーセント計算 |

## 🚀 GitHub Pages へのデプロイ方法

本リポジトリには、GitHub Actions を使用して自動的に GitHub Pages へデプロイするワークフロー（[.github/workflows/deploy.yml](.github/workflows/deploy.yml)）があらかじめ設定されています。

### 設定手順

1. このソースコードを GitHub リポジトリにプッシュします。
2. GitHub 上の対象リポジトリページを開き、上部タブの **Settings** に進みます。
3. 左サイドバーの **Pages** を選択します。
4. **Build and deployment** セクションの **Source** ドロップダウンで **GitHub Actions** を選択します。
5. ソースコードを `main` ブランチにプッシュ（または Settings から Actions ワークフローを手動実行）すると、自動的にビルドとデプロイが行われ、公開URLが生成されます。
